# 9tsu-tag-fetcher
Command line client for 9tsu tag video mp3 list up

## Requirements

 - Node 7.6 or later

## Installation

```bash
npm i -g 9tsu-tag-fetcher
```

## Usage
```bash                                                                                     
  Usage: 9tsu-tag-fetcher <tag>                                                                                    
                                                                                                                         
                                                                                                                               
                                                                                                                               
  Options:                                                                                                                     
                                                                                                                               
    -V, --version     output the version number
    -h, --help        output usage information  
```

## Example
```bash
9tsu-tag-fetcher new
```
